The user wants to create a global mobile trivia app called “Knowledge Wars”.

**PRODUCT REQUIREMENTS:**
- **UI/UX:** A super modern, novel interface that appeals to both youth and discerning users, with a cyberpunk/neon gaming aesthetic.
- **Core Gameplay:** A real-time 1v1 PvP trivia game. Users challenge each other, pick a topic, and answer 10 questions. The first to answer correctly gets +2 points. A wrong answer blocks the player for that question. An optional hint costs -1 point. The player with the most points after 10 questions wins.

- **Payments:** Must use native In-App Purchases (Apple StoreKit 2, Google Play Billing), not Stripe. Backend must handle server-side receipt validation.
- **Matchmaking:** Registered users only. A global random pool with an ELO rating system. Users can be challenged via push notifications.
- **Topics & Questions:** Supports dynamic user-created topics (with moderation) and predefined categories. Questions are to be generated by OpenAI's  model with robust caching.
- **Leaderboards & Ranking:** ELO-based leagues (Bronze to Grand Master) and various leaderboards (global, weekly, by topic).
- **Technology:**
    - **App:** React Native or Flutter (developed as a responsive web app for now).
    - **Backend:** FastAPI with WebSockets.
    - **Database:** MongoDB (used in dev), with a plan for PostgreSQL or equivalent.
    - **Real-time:** WebSockets for duel state.
    - **Push Notifications:** FCM/APNs.
